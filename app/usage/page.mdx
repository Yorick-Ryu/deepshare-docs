# 使用教程

本教程将详细介绍 DeepShare 的各项功能使用方法。

## 基本使用流程

### 前提条件

- 已安装 DeepShare 插件
- 已刷新 AI 对话页面

### 快速开始

1. 访问支持的 AI 平台（如 https://chat.deepseek.com/）
2. 开始一个新对话
3. 在页面右上角可以看到 DeepShare 添加的按钮
4. 选择您需要的功能

## 功能详解

### 📄 导出为 Word 文档

#### DeepSeek / ChatGPT / Gemini（一键导出）

对于 DeepSeek、ChatGPT 和 Gemini，DeepShare 提供一键导出功能：

**步骤：**

1. 在对话页面找到"导出为 Word"按钮
2. 点击按钮
3. 等待转换完成（通常需要几秒钟）
4. 自动下载 Word 文档

**注意事项：**

- 首次使用需要配置 API Key（[购买 API Key](/pricing)）
- 每次转换消耗 1 次配额
- 支持公式、表格、代码块等复杂格式

#### 其他平台（手动粘贴）

对于豆包、元宝、Kimi 等平台，使用手动粘贴方式：

**步骤：**

1. 在 AI 回答末尾找到复制按钮（通常是"❐"图标）
2. 点击复制按钮，复制 Markdown 格式的回答
3. 点击 DeepShare 插件图标，打开转换器
4. 将复制的内容粘贴到输入框
5. 点击"转换为文档"按钮
6. 下载生成的 Word 文档

**在线转换器：**

您也可以使用在线转换器：https://ds.rick216.cn/converter.html

### 🔢 复制 LaTeX 公式

#### 使用方法

1. 在 AI 的回答中找到数学公式
2. 将鼠标移到公式上，公式会高亮显示
3. 点击公式
4. 公式的 LaTeX 代码会自动复制到剪贴板
5. 显示"已复制"提示

#### 支持的格式

**LaTeX 格式**（默认）：
```latex
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
```

**MathML 格式**：
可在插件设置中切换到 MathML 格式。

#### 粘贴到 Word

**方法一：直接粘贴 LaTeX**
1. 在 Word 中按 Ctrl+V 粘贴
2. 选择"只保留文本"
3. 选中粘贴的 LaTeX 代码
4. 点击 Word 的"插入" → "公式" → "LaTeX"

**方法二：使用 Word 公式编辑器**
1. 在 Word 中插入新公式
2. 点击"LaTeX"按钮
3. 粘贴复制的 LaTeX 代码
4. Word 会自动渲染公式

### 📸 对话截图（DeepSeek）

#### 截取所有对话

**步骤：**
1. 点击页面右上角的"分享"按钮（相机图标）
2. 选择截图选项
3. 等待截图生成
4. 下载或复制图片

#### 选择性截图

**步骤：**
1. 点击"选择对话"按钮（复选框图标）
2. 勾选想要截图的对话
3. 使用"全选"/"取消全选"快速操作
4. 点击"分享"按钮
5. 选择截图选项
6. 下载或复制图片

#### 分享格式选择

**截图模式**：
- 生成高质量长图
- 适合社交媒体分享
- 保留原始排版效果

**文本模式**：
- 纯文本格式
- 方便复制粘贴
- 体积小，传输快

## 水印设置

### 自定义水印

**步骤：**
1. 点击分享对话框右上角的"设置"图标（齿轮）
2. 在"水印设置"区域：
   - 输入自定义水印文本
   - 或勾选"隐藏水印"选项
3. 关闭设置窗口
4. 水印设置会自动保存

**水印示例：**
- `@用户名` - 添加您的社交账号
- `Made with DeepShare` - 推广工具
- 或任何您想要的文本

### 隐藏水印

如果您不需要水印：
1. 在设置中勾选"隐藏水印"
2. 生成的截图将不包含任何水印

## 配置 API Key

### 获取 API Key

1. 访问 [购买页面](/pricing)
2. 选择合适的套餐
3. 完成支付
4. 通过邮件接收 API Key

### 配置步骤

**方法一：通过插件弹窗**
1. 点击浏览器工具栏的 DeepShare 图标
2. 在"Word 转换设置"部分：
   - 服务器地址：`https://api.ds.rick216.cn`
   - API Key：粘贴您的 API Key
3. 点击"保存"

**方法二：通过在线转换器**
1. 访问 https://ds.rick216.cn/converter.html
2. 滚动到"转换设置"部分
3. 填入 API Key
4. 设置会自动同步到浏览器插件

### 查询剩余次数

**通过插件**：
插件弹窗会显示您的剩余转换次数。

**通过在线工具**：
1. 访问 [购买页面](/pricing)
2. 在"查询剩余次数"部分输入 API Key
3. 点击"查询配额"
4. 查看详细的配额信息

## 使用技巧

### 提高 Word 导出质量

1. **完整的对话**：确保对话内容完整，避免截断
2. **格式规范**：使用标准 Markdown 格式
3. **公式检查**：导出前检查公式显示是否正确
4. **表格对齐**：复杂表格建议手动调整

### 批量处理对话

如果需要导出多个对话：
1. 分别打开每个对话
2. 逐个导出为 Word
3. 或将多个对话内容复制合并后一起转换

### 管理 API 配额

- 查看每月使用情况
- 计划性使用转换次数
- 及时续费避免配额不足

## 故障排除

### Word 导出失败

**可能原因**：
- API Key 错误或过期
- 配额已用完
- 网络连接问题
- 对话内容格式问题

**解决方法**：
1. 检查 API Key 是否正确
2. 查询剩余配额
3. 检查网络连接
4. 简化对话内容重试

### 公式复制不工作

**可能原因**：
- 页面未刷新
- 插件未启用
- 平台不支持

**解决方法**：
1. 刷新页面（F5）
2. 检查插件是否启用
3. 确认平台在支持列表中

### 截图生成失败

**可能原因**：
- 对话内容过长
- 浏览器内存不足
- 不支持的平台

**解决方法**：
1. 分段截图
2. 关闭其他标签页
3. 只在 DeepSeek 使用截图功能

# 去除Word中的分割线

<Callout>

请确保浏览器已安装 DeepShare 插件，未安装请查看[安装指南](/installation)

本功能需要购买API-KEY，[点此查看定价](/pricing)

</Callout>

本功能可以去除导出的Word中的分割线

<Steps>

### 点击工具栏中的DeepShare插件

如果没找到的话请先[在工具栏中固定DeepShare](/installation#在工具栏中固定deepshare)
![Click DeepShare](/images/deepshare/click-deepshare.png)

### 勾选「去除分割线」

![Remove Divoder](/images/deepshare/remove-divider.png)

</Steps>

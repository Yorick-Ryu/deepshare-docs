# 功能介绍

DeepShare 提供了一系列强大的功能，帮助您更好地管理和分享 AI 对话内容。

## 核心功能

### 📄 导出为Word文档

将 AI 对话一键转换为排版精美的Word文档。

**特点：**
- 完美保留公式格式（LaTeX 数学公式）
- 保留表格结构和格式
- 支持代码块高亮
- 自动处理图片和链接
- 精美的排版效果

**支持平台：**
- DeepSeek（一键导出）
- ChatGPT（一键导出）
- Gemini（一键导出）
- 其他平台（手动粘贴转换）

### 🔢 LaTeX公式复制

点击 AI 对话中的任意数学公式即可快速复制。

**特点：**
- 一键复制 LaTeX 代码
- 支持 MathML 格式
- 轻松粘贴到 Word、LaTeX 编辑器等
- 支持行内公式和块级公式

**支持平台：**
- DeepSeek ✓
- ChatGPT ✓
- 豆包 ✓
- Grok ✓
- 通义千问 ✓
- 讯飞星火 ✓
- 智谱清言 ✓
- OpenRouter ✓
- Poe ✓
- Monica ✓
- 问小白 ✓
- AskManyAI ✓
- 万知 ✓
- AI智慧岛 ✓
- 纳米AI ✓
- 知乎 ✓

### 📸 对话长截图

轻松截取完整的对话内容，生成高质量图片。

**特点：**
- 支持长对话完整截图
- 可选择性分享对话内容
- 支持自定义水印
- 一键全选/取消全选
- 高清图片输出

**支持平台：**
- DeepSeek（专属功能）

### 🎨 自定义水印

在截图中添加您的个性化水印。

**功能：**
- 自定义水印文本
- 可选择隐藏默认水印
- 灵活的水印位置

## 隐私保护

DeepShare 高度重视您的隐私：

- **零数据收集**：不收集或存储您的个人信息
- **本地处理**：所有内容处理均在您的设备本地完成
- **无行为跟踪**：不跟踪您的浏览行为
- **数据加密**：文档转换服务采用加密传输
- **即时删除**：转换完成后立即删除所有数据

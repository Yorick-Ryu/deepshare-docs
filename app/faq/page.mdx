# 常见问题 FAQ

## 安装与配置

### 如何安装 DeepShare？

**从应用商店安装**：
   - [Edge 外接程序商店](https://microsoftedge.microsoft.com/addons/detail/deepshare/pdccjnppfegekpnhfljbngammgfbcofm)
   - [Chrome 网上应用商店](https://chromewebstore.google.com/detail/omnaecaamcabmnbjnpjpecoaalfgidop)
   - [Firefox 附加组件](https://addons.mozilla.org/firefox/addon/deepshare/)

详细步骤请查看[安装指南](/installation)。

### 安装后看不到按钮怎么办？

**原因与解决方法：**

1. **未刷新页面**：安装后需要刷新已打开的 AI 对话页面（按 F5）
2. **插件未启用**：检查浏览器扩展管理页面，确保 DeepShare 已启用
3. **平台不支持**：确认您访问的 AI 平台在[支持列表](/#支持的平台)中
4. **浏览器兼容性**：确保使用的是最新版本的Edge、Chrome或Firefox浏览器

### 如何配置 API Key？

**步骤：**

1. [购买 API Key](/pricing)
2. 点击浏览器工具栏的 DeepShare 图标（蓝色小鸟）
3. 粘贴您的 API Key

## 功能使用

### 哪些平台支持一键导出 Word？

目前支持一键导出的平台：

- ✅ DeepSeek
- ✅ ChatGPT
- ✅ Gemini

其他平台支持手动粘贴方式转换。查看[完整支持列表](/platforms)

### 如何在豆包等平台导出 Word？

对于不支持一键导出的平台，使用手动粘贴方式：

1. 在 AI 回答末尾点击复制按钮（❐）
2. 打开 DeepShare 插件弹窗
3. 粘贴复制的内容
4. 点击"转换为文档"
5. 下载生成的 Word 文档

也可以使用[在线转换器](https://ds.rick216.cn/converter.html)

详细教程请查看[使用教程 - 豆包导出Word](/usage/doubao-to-word)

### 如何复制公式？

请查看[使用教程 - 复制公式](/usage/copy-formula)

## 技术问题

### Word 导出失败怎么办？

**常见原因和解决方法：**

1. **API Key 错误**
   - 检查 API Key 是否正确
   - 确认没有多余的空格

2. **配额不足**
   - 查询剩余次数
   - 及时续费

3. **网络问题**
   - 检查网络连接
   - 尝试使用 VPN（如果需要）

4. **内容格式问题**
   - 简化对话内容
   - 删除特殊字符
   - 分段转换

### 公式复制不工作？

**解决方法：**

1. **刷新页面**：按 F5 刷新 AI 对话页面
2. **检查插件**：确认 DeepShare 已启用
3. **平台支持**：确认该平台支持公式复制
4. **重新安装**：卸载后重新安装插件

### 截图生成失败？

**可能原因：**

1. **对话过长**：尝试分段截图
2. **内存不足**：关闭其他标签页
3. **平台限制**：只有 DeepSeek 支持截图

### 插件图标不显示？

**解决方法：**

1. 检查插件是否固定在工具栏
2. 检查插件是否已安装
3. 在扩展管理页面启用插件
4. 重启浏览器
5. 重新安装插件

### 导出的 Word 格式不对？

**优化方法：**

1. **公式问题**：确保原始公式格式正确
2. **表格问题**：复杂表格可能需要手动调整
3. **代码问题**：检查代码块格式
4. **使用反馈**：[联系我们](/contact)报告问题

### Firefox 提示安全警告？

这是正常现象，临时安装的扩展会有此提示。

**解决方法：**
- 点击"继续"
- 或从官方 Firefox 附加组件商店安装

**不会！** DeepShare 坚持零数据收集原则：

- ❌ 不收集个人信息
- ❌ 不跟踪浏览行为
- ❌ 不使用分析工具
- ✅ 所有处理均在本地完成
- ✅ Word 转换数据即时删除

详细信息请查看[隐私政策](/privacy)。

### API Key 安全吗？

**安全措施：**

- 🔐 加密存储在浏览器本地
- 🔐 采用 HTTPS 加密传输
- 🔐 不在服务器持久化存储

**安全建议：**
- 不要与他人共享 API Key
- 定期检查使用情况
- 发现异常立即联系我们

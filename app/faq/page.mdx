# 常见问题 FAQ

## 安装与配置

### 如何安装 DeepShare？

DeepShare 支持三种安装方式：

1. **从应用商店安装（推荐）**：
   - [Edge 外接程序商店](https://microsoftedge.microsoft.com/addons/detail/deepshare/pdccjnppfegekpnhfljbngammgfbcofm)
   - [Chrome 网上应用商店](https://chromewebstore.google.com/detail/omnaecaamcabmnbjnpjpecoaalfgidop)
   - [Firefox 附加组件](https://addons.mozilla.org/firefox/addon/deepshare/)

2. **从源代码安装**：
   - 下载 [GitHub 源代码](https://github.com/Yorick-Ryu/deep-share)
   - 在浏览器扩展页面启用开发者模式
   - 加载已解压的扩展程序

详细步骤请查看[安装指南](/installation)。

### 安装后看不到按钮怎么办？

**原因与解决方法：**

1. **未刷新页面**：安装后需要刷新已打开的 AI 对话页面（按 F5）
2. **插件未启用**：检查浏览器扩展管理页面，确保 DeepShare 已启用
3. **平台不支持**：确认您访问的 AI 平台在[支持列表](/#支持的平台)中
4. **浏览器兼容性**：确保使用的是最新版本的浏览器

### 如何配置 API Key？

**步骤：**

1. [购买 API Key](/pricing)
2. 点击浏览器工具栏的 DeepShare 图标
3. 填入服务器地址：`https://api.ds.rick216.cn`
4. 粘贴您的 API Key
5. 点击保存

详细教程请查看[使用教程 - 配置 API Key](/usage#配置-api-key)。

## 功能使用

### 哪些平台支持一键导出 Word？

目前支持一键导出的平台：

- ✅ DeepSeek
- ✅ ChatGPT
- ✅ Gemini

其他平台支持手动粘贴方式转换。查看[完整支持列表](/platforms)。

### 如何在豆包等平台导出 Word？

对于不支持一键导出的平台，使用手动粘贴方式：

1. 在 AI 回答末尾点击复制按钮（❐）
2. 打开 DeepShare 插件弹窗
3. 粘贴复制的内容
4. 点击"转换为文档"
5. 下载生成的 Word 文档

也可以使用[在线转换器](https://ds.rick216.cn/converter.html)。

### 如何复制公式？

**步骤：**

1. 在 AI 回答中找到数学公式
2. 将鼠标移到公式上（公式会高亮）
3. 点击公式
4. LaTeX 代码自动复制到剪贴板
5. 粘贴到您需要的地方

支持的格式：LaTeX（默认）、MathML。

### 如何截取 DeepSeek 对话？

**完整截图：**
1. 点击页面右上角的"分享"按钮
2. 等待截图生成
3. 下载图片

**选择性截图：**
1. 点击"选择对话"按钮
2. 勾选想要截图的对话
3. 点击"分享"按钮
4. 下载图片

详细教程请查看[使用教程](/usage)。

### 如何添加自定义水印？

**步骤：**

1. 点击分享对话框右上角的"设置"图标
2. 在"水印设置"中输入自定义文本
3. 或勾选"隐藏水印"不显示水印
4. 设置自动保存

## 技术问题

### Word 导出失败怎么办？

**常见原因和解决方法：**

1. **API Key 错误**
   - 检查 API Key 是否正确
   - 确认没有多余的空格

2. **配额不足**
   - 查询剩余次数
   - 及时续费

3. **网络问题**
   - 检查网络连接
   - 尝试使用 VPN（如果需要）

4. **内容格式问题**
   - 简化对话内容
   - 删除特殊字符
   - 分段转换

### 公式复制不工作？

**解决方法：**

1. **刷新页面**：按 F5 刷新 AI 对话页面
2. **检查插件**：确认 DeepShare 已启用
3. **平台支持**：确认该平台支持公式复制
4. **重新安装**：卸载后重新安装插件

### 截图生成失败？

**可能原因：**

1. **对话过长**：尝试分段截图
2. **内存不足**：关闭其他标签页
3. **平台限制**：只有 DeepSeek 支持截图

### 插件图标不显示？

**解决方法：**

1. 检查插件是否已安装
2. 在扩展管理页面启用插件
3. 重启浏览器
4. 重新安装插件

### 导出的 Word 格式不对？

**优化方法：**

1. **公式问题**：确保原始公式格式正确
2. **表格问题**：复杂表格可能需要手动调整
3. **代码问题**：检查代码块格式
4. **使用反馈**：联系我们报告问题

### Firefox 提示安全警告？

这是正常现象，临时安装的扩展会有此提示。

**解决方法：**
- 点击"继续"
- 或从官方 Firefox 附加组件商店安装

## 隐私与安全

### DeepShare 会收集我的数据吗？

**不会！** DeepShare 坚持零数据收集原则：

- ❌ 不收集个人信息
- ❌ 不跟踪浏览行为
- ❌ 不使用分析工具
- ✅ 所有处理均在本地完成
- ✅ Word 转换数据即时删除

详细信息请查看[隐私政策](/privacy)。

### API Key 安全吗？

**安全措施：**

- 🔐 加密存储在浏览器本地
- 🔐 采用 HTTPS 加密传输
- 🔐 不在服务器持久化存储

**安全建议：**
- 不要与他人共享 API Key
- 定期检查使用情况
- 发现异常立即联系我们

### 对话内容会被保存吗？

**Word 转换服务：**
- 仅临时处理您选择的内容
- 转换完成后**立即删除**
- 不用于模型训练或其他目的

**其他功能：**
- 完全在本地浏览器处理
- 不涉及任何数据上传

## 如何购买

1. 访问 [购买页面](https://ds.rick216.cn/purchase.html)
2. 填写您的邮箱（用于接收 API Key）
3. 选择充值金额或输入自定义金额
4. 完成支付
5. 通过邮件接收 API Key

## 如何使用

完成购买后，按照以下步骤设置您的 DeepShare 扩展：

1. 在浏览器中点击 DeepShare 扩展图标
2. 填入您收到的 API Key
3. 保存设置

现在您可以在 AI 对话中使用「导出为 Word」功能了！

## 查询剩余次数

如果您已购买转换次数，可以查询您的配额情况：

### 通过在线工具

1. 访问 [购买页面](https://ds.rick216.cn/purchase.html)
2. 在"查询剩余次数"部分输入您的 API Key
3. 点击"查询配额"按钮
4. 查看详细的配额信息：
   - 总计次数
   - 已使用次数
   - 剩余次数
   - 有效期至

### 通过插件

点击浏览器工具栏的 DeepShare 图标，插件弹窗会显示您的剩余转换次数。

## 续费说明

当您的配额即将用完或到期时：

1. 访问 [续费页面](https://ds.rick216.cn/renew.html)
2. 输入您现有的 API Key
3. 选择续费套餐
4. 完成支付
5. 次数将自动添加到您的账户

**注意**：续费会延长有效期，并增加转换次数。

1. 访问 [购买页面](https://ds.rick216.cn/purchase.html)
2. 填写您的邮箱（用于接收 API Key）

## 如何使用

完成购买后，按照以下步骤设置您的 DeepShare 扩展：

1. 在浏览器中点击 DeepShare 扩展图标
2. 填入服务器地址：`https://api.ds.rick216.cn`
3. 填入您收到的 API Key
4. 保存设置

现在您可以在 AI 对话中使用「导出为 Word」功能了！

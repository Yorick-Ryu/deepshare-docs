# 安装指南

本指南将帮助您在不同浏览器上安装 DeepShare 插件。

## 从应用商店安装（推荐）

### Microsoft Edge

1. 访问 [Edge 外接程序商店](https://microsoftedge.microsoft.com/addons/detail/deepshare/pdccjnppfegekpnhfljbngammgfbcofm)
2. 点击"获取"按钮
3. 在弹出的对话框中确认安装
4. 安装完成后，浏览器工具栏会显示 DeepShare 图标

### Google Chrome

1. 访问 [Chrome 网上应用商店](https://chromewebstore.google.com/detail/omnaecaamcabmnbjnpjpecoaalfgidop)（*国内访问受限*）
2. 点击"添加至 Chrome"按钮
3. 在弹出的对话框中确认安装
4. 安装完成后，浏览器工具栏会显示 DeepShare 图标

### Mozilla Firefox

1. 访问 [Firefox 附加组件商店](https://addons.mozilla.org/firefox/addon/deepshare/)
2. 点击"添加到 Firefox"按钮
3. 在弹出的对话框中确认安装并授予权限
4. 安装完成后，浏览器工具栏会显示 DeepShare 图标

## 本地安装（开发者模式）

如果您想使用最新的开发版本，可以从源代码安装。

### 下载源代码

从 GitHub 下载最新版本：

```bash
git clone https://github.com/Yorick-Ryu/deep-share.git
```

或者直接下载 ZIP 文件：
[下载 DeepShare 源代码](https://github.com/Yorick-Ryu/deep-share/releases)

### Edge / Chrome 安装步骤

1. 解压下载的源代码文件
2. 打开浏览器，在地址栏输入：
   - Edge: `edge://extensions/`
   - Chrome: `chrome://extensions/`
3. 启用右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择解压后的 `deep-share` 文件夹
6. 安装完成！

## 安装后的设置

### 刷新页面

⚠️ **重要提示**：安装插件后，请刷新所有已打开的 AI 对话页面，以使插件生效。

### 配置 Word 转换功能

如果您需要使用 Word 文档转换功能：

1. 点击浏览器工具栏的 DeepShare 图标
2. 在弹出的窗口中，找到"文档转换设置"
3. 填入您的 API Key（[购买 API Key](/pricing)）

## 验证安装

安装完成后，您可以通过以下方式验证：

### 检查浏览器工具栏

浏览器工具栏应该显示 DeepShare 的图标（一只蓝色小鸟）

### 访问 AI 平台

1. 打开 [DeepSeek](https://chat.deepseek.com/)
2. 开始一个新对话
3. 在页面右上角应该能看到 DeepShare 添加的按钮

### 测试公式复制

1. 让 AI 生成一个包含数学公式的回答
2. 尝试点击公式
3. 应该能看到复制成功的提示

## 常见安装问题

### 插件图标不显示

**解决方法**：
1. 确认插件已成功安装
2. 刷新浏览器页面
3. 如果使用开发者模式安装，检查是否有错误提示

### 在 AI 页面看不到按钮

**解决方法**：
1. 刷新 AI 对话页面（F5）
2. 检查插件是否已启用
3. 确认访问的 AI 平台在支持列表中

### Firefox 提示安全警告

**解决方法**：
1. 这是正常现象，临时安装的扩展会有此提示
2. 点击"继续"即可
3. 建议从官方 Firefox 附加组件商店安装

### Edge/Chrome 安装失败

**解决方法**：
1. 确认已启用"开发者模式"
2. 检查文件夹路径是否正确
3. 查看浏览器控制台的错误信息
4. 尝试重新下载源代码

## 更新插件

### 应用商店版本

从应用商店安装的版本会自动更新，无需手动操作。

### 本地安装版本

如果使用开发者模式安装：

1. 下载最新的源代码
2. 删除旧版本文件
3. 按照安装步骤重新安装

或者使用 Git 更新：

```bash
cd deep-share
git pull origin main
```

然后在浏览器扩展页面点击"重新加载"按钮。

## 卸载插件

如果您需要卸载 DeepShare：

### Edge / Chrome

1. 打开扩展页面
2. 找到 DeepShare
3. 点击"移除"按钮
4. 确认卸载

### Firefox

1. 打开 `about:addons`
2. 找到 DeepShare
3. 点击"移除"按钮
4. 确认卸载

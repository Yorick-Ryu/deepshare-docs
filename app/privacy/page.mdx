# 隐私政策

## 1. 数据收集与处理原则

DeepShare 是一个以用户隐私保护为核心的开源浏览器插件，我们坚持以下原则：

- **零数据收集**：我们不会收集或存储您的任何个人信息
- **本地处理**：所有内容处理均在您的设备本地完成
- **无行为跟踪**：我们不会跟踪您的浏览行为或使用任何分析工具
- **无第三方共享**：您的数据不会与任何第三方共享

## 2. 功能隐私说明

### 2.1 剪贴板数据处理

在使用本工具时：

- 仅在您**明确授权**的情况下，DeepShare 才会读取剪贴板中的内容
- 剪贴板内容仅用于格式转换
- 我们**不会**存储您的任何剪贴板数据到任何服务器

### 2.2 对话截图和公式复制功能

所有操作均在您的浏览器本地进行：

- 截图功能仅在您的浏览器内执行，不涉及数据上传
- 对话内容仅临时保存在浏览器内存中，不会持久化存储
- 导出的图片和文本直接保存到您的设备，不经过任何第三方服务器

### 2.3 Word 文档转换功能

在使用文档转换服务时：

- 我们**仅处理**您选择的内容，且仅用于提供文档转换服务
- 数据在转换完成后**立即删除**，不会保留任何副本
- 数据传输采用**加密**方式，确保安全
- 我们**绝不会**将您的数据用于训练模型或其他目的

## 3. 用户数据权利

作为 DeepShare 的用户，您拥有以下权利：

- **透明了解**：随时了解我们如何处理您的数据
- **不被跟踪**：享受无跟踪的使用体验
- **数据控制**：所有数据操作均由您控制，我们不会在未经您同意的情况下访问任何数据

## 4. 隐私保护措施

我们采取以下技术措施保护您的隐私：

- **本地处理优先**：尽可能在您的设备本地完成数据处理
- **最小数据原则**：仅处理完成功能所必需的最少数据
- **加密传输**：任何必要的数据传输均采用加密方式
- **及时清除**：处理完成后立即清除所有数据

## 5. 数据安全

### 5.1 传输安全

- 使用 HTTPS 加密协议传输数据
- 采用行业标准的加密算法
- 定期更新安全措施

### 5.2 存储安全

- 本地数据存储在浏览器的安全存储区域
- API Key 等敏感信息经过加密存储
- 不在服务器端存储任何用户数据

## 6. Cookie 和本地存储

DeepShare 使用浏览器的本地存储功能来保存您的设置：

- **存储内容**：插件配置、API Key、用户偏好设置
- **存储位置**：您的浏览器本地
- **访问权限**：仅 DeepShare 插件可以访问
- **清除方式**：可通过浏览器设置清除

我们**不使用** Cookie 进行跟踪或分析。

## 7. 开源透明

DeepShare 是一个开源项目：

- 源代码公开在 [GitHub](https://github.com/Yorick-Ryu/deep-share)
- 任何人都可以审查代码
- 欢迎安全研究人员审计
- 接受社区的隐私保护建议

## 8. 儿童隐私

DeepShare 不针对 13 岁以下的儿童。我们不会故意收集儿童的个人信息。

## 9. 国际用户

DeepShare 可供全球用户使用。无论您在哪里，我们都遵守相同的隐私保护标准。

## 10. 隐私政策更新

如果我们对隐私政策进行任何更改：

- 我们将在此页面上发布更新的版本
- 更新修改日期
- 重大变更会通过 GitHub 公告通知用户

我们建议您定期查看本隐私政策以了解最新信息。

## 11. 数据删除

如果您想删除 DeepShare 存储的所有数据：

### 删除本地数据

1. 卸载 DeepShare 插件
2. 或在浏览器扩展设置中清除插件数据

### 删除服务器数据

由于我们不在服务器上持久化存储任何用户数据，因此无需额外操作。

### 删除 API Key

如果您想注销 API Key：

1. 联系我们请求删除
2. 我们会在 7 个工作日内处理

## 12. 您的权利

根据相关隐私法规，您拥有以下权利：

- **访问权**：了解我们如何处理您的数据
- **更正权**：更正不准确的个人信息
- **删除权**：请求删除您的数据
- **反对权**：反对特定的数据处理方式
- **数据携带权**：以结构化格式获取您的数据

要行使这些权利，请联系我们。
